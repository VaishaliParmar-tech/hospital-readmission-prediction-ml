How It Works

A complete machine learning pipeline from raw data to actionable predictions.

01

Data Collection

Hospital records of 2,200 diabetic patients with 17 clinical features including lab results, diagnoses, and demographics.

02

Preprocessing

Missing value imputation (median/mode), IQR outlier capping, and label encoding for categorical variables.

03

Model Training

Random Forest as the primary model with 7 comparative classifiers. All evaluated with 5-fold cross-validation.

04

Prediction

Enter patient data, get instant readmission risk prediction with probability score and risk classification.

Key Features

Random Forest — Primary Model

Our main model uses a tuned Random Forest classifier with 300 trees, max depth 20, sqrt features, and bootstrap sampling — achieving superior accuracy and interpretability.
