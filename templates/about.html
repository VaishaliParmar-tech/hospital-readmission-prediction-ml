About This Project

A Machine Learning approach to predict hospital readmission in diabetic patients — built with passion by our team.

Project Overview

Hospital Readmission Prediction is a supervised machine learning project that aims to predict whether a diabetic patient will be re-admitted to a hospital within 30 days of discharge. Early identification of high-risk patients enables hospitals to allocate resources effectively, improve care quality, and reduce preventable readmissions.

The dataset contains 2,200 records of diabetic patients with 17 clinical features including age, duration of hospital stay, number of lab procedures, medications, diagnoses, and glucose/A1C test results.

Our primary model is a Random Forest Classifier, chosen for its excellent performance on tabular medical data, resistance to overfitting, and inherent feature importance analysis. Six other classifiers are trained for comparative benchmarking — all evaluated with 5-fold cross-validation.

⚙️ Methodology
01

Data Loading

Patient hospital records loaded from Excel/CSV. 17 clinical features for 2,200 patients with binary readmission target.

02

Preprocessing

Median imputation for numerical columns, mode imputation for categorical. IQR-based outlier capping to clip extreme values.

03

Encoding

Label encoding applied to all categorical variables (age, diagnoses, test results, etc.) using scikit-learn LabelEncoder.

04

Train/Test Split

80/20 stratified train-test split with random_state=42 for reproducibility. 1,760 training, 440 test samples.

05

Model Training

Random Forest primary model + 6 comparative models (LR, DT, SVM, KNN, AdaBoost, Naive Bayes). CV=5 evaluation.

06

Evaluation

Accuracy, ROC-AUC, Confusion Matrix, Feature Importance, Overfitting analysis visualised across all models.

🎓 Project Guide

This project was developed as part of our curriculum on Machine Learning and Data Science. We applied real-world medical data to build a complete end-to-end ML pipeline — from raw data preprocessing to web deployment — with a focus on model interpretability and clinical relevance.

The primary focus is on Random Forest as our main classifier, with GridSearchCV tuning and 5-fold cross-validation ensuring robust and generalizable results.
